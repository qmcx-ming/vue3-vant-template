<h1 align="center" style="margin: 30px 0 30px; font-weight: bold;">Vue3-Vant-Template</h1>
<h4 align="center">
基于Vue3+Vant4的移动端H5模板【没有TS😋】
</h4>

## 使用

```sh
# 安装依赖
npm install

# 运行
npm run dev

# 打包
npm run build

# 预览
npm run preview
```
